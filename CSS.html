CSS Basics

Welcome

Welcome to CSS Basics. This workshop leads you through the basics of Cascading Style Sheets (CSS). CSS is the styling language for web pages. You will learn to use CSS to style HTML elements and control the presentation of your web content.

Objectives

  • Understand the basic syntax of CSS rules (selectors and declarations).
  • Be able to use different types of CSS selectors (element, class, ID).
  • Apply styles to elements using CSS properties (e.g., color, font, margin, padding).
  • Understand the CSS box model (content, padding, border, margin).
  • Learn how to use external stylesheets to separate CSS from HTML.
  • Understand CSS specificity and how styles are applied.

Prerequisites

You should have a basic understanding of HTML. A text editor and a web browser are also required.

CSS Example

body {
  background-color: #f0f0f0;
  font-family: sans-serif;
}

h1 {
  color: navy;
  text-align: center;
}

p {
  line-height: 1.6;
}

Save this code as styles.css and link it to your HTML file using <link rel="stylesheet" href="styles.css"> in the <head> section.
